# AGENTS

## Project Shape

- This is a small browser-first app with no build step.
- Keep the core experience in `index.html`, `styles.css`, and `app.js`.
- Preserve the historic-photo overlay workflow: load reference, align camera, capture, compare, download.

## Working Rules

- Prefer plain HTML, CSS, and JavaScript unless a native feature truly requires Capacitor.
- Keep the UI mobile-first and readable outdoors.
- Use additive edits for the docs: `README.md`, `CHANGELOG.md`, `DECISIONS.md`, and `TODO.md`.
- Avoid introducing dependencies unless the app clearly needs them.
